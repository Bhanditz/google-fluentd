This will install Google Fluentd on your Mac. -------------------------------------------------- You can start / stop Google Fluentd via launchctl command. - start google-fluentd % sudo launchctl load /Library/LaunchDaemons/google-fluentd.plist - stop google-fluentd % sudo launchctl unload /Library/LaunchDaemons/google-fluentd.plist If you have a question, please ask it on Fluentd ML https://groups.google.com/forum/#!forum/fluentd
